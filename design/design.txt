次元碎片
====================
也许是更高维度发生了争斗或什么原因，我们发现了一些次元碎片，这些碎片具有超越我们所处维度的力量，似乎可以操作时间和空间...

次元科技线以后期为主，贵，慢，但比较强。

后面会有孢子科技线、传统科技线、魔法科技线，还没设计。

## 物品
- 次元碎片
    - 爆钻采集
    - 设定上比较硬，有玻璃特性
    - 作为弹药击退很强

- 空间结晶
    - 次元碎片 + 硅 + 钍 + 矿渣
    - 可控的压缩与膨胀

- 时间结晶
    - 次元碎片 + 塑钢 + 相织 + 冷却液
    - 可控的物质活性

- 硬钍合金
    - 空间结晶 + 钍 + 石墨，用于围墙和弹药
    - 硬的要命，可能是被不可逆压缩了

- 时空合金
    - 空间结晶 + 时间结晶 + 巨浪合金 + 离子载液
    - 能炸
    - 能放电
    - 具有空间和时间结晶的特性

## 液体
- 时间流（时间结晶 + 砂子 + 冷冻液，用于冷却和超速）
- 离子载液（时间流 + 巨浪合金，用于武器）

## 炮台
### 原版更改
- 为分裂炮、气旋、浪涌，增加空间碎片弹药，接近塑钢的伤害，但击退很高
- 为连射、幽灵增加硬钍合金弹药，伤害约等于 钍 * 1.5
- 为噩兆增加空间结晶弹药，具有黑洞效果
- 为噩兆增加时空合金弹药，具有光棱塔效果（太屌取消）
- 为气旋增加时空合金弹药，类似更屌的巨浪（太屌取消）
- 滋水枪增加两种液体，离子载液伤害高一点，达到合金气旋差不多的水平

### 新炮
- 电子团炮（3级，发射‘小型电子团’）
- 离子炮（4级，类似 TAU 子弹，爆炸范围较大）
- 重型离子激光炮（5级，使用离子载液，末端爆炸的激光，中间出现少量小型电子团）
- 重型电子团炮（5级，射击时空合金，发射大型电子团）

## 钻头
- 硬钍钻头，加大量水超级快

## 运输
- 次元传送桥（基础材料 + 相织 + 次元碎片，无需次元核心）
- 压缩传送带（塑钢带 + 空间结晶）
- 超空间提取器（提取器 + 相织 + 时空合金）
- 超空间推送器（时空合金 贵就完了，工厂革命）

## 液体
- 硬钍路由器（硬，防火）
- 硬钍导管（硬，防火）
- 空间储液罐
- 极速泵（时间结晶）

## 防御
- 硬钍围墙
- 次元相织墙

## 工厂
- 空间结晶器（热就完了）
- 时间结晶器（冷就完了）
- 硬钍冶炼厂
- 时空冶炼厂
- 时间凝结机
- 离子工厂
- 时空结晶器（贵，多合成表）

## 单位
T4 T5 时空单位工厂，可以从原版单位 T3 转化为时空单位 T4，以此类推

### 陆地辅助线
- T4 提供血量大约等于绿色t3 的反射护盾，使用有修复效果的轻型离子炮
- T5 使用削弱的重型离子激光炮，射程比绿的近，瞬发

### 陆地进攻线（爬虫也先算进攻）
- T4 使用重型电子团
- T5 使用重型离子炮（非激光）

### 空中辅助线
暂无

### 空中进攻线
- T4 电子团
- T5 大量电子团 + 离子炮，跳刀 + 短时间反弹所有的护盾

### 海军线
暂无

## 其他
- 时间过载（更屌的超速）
- 空间仓库（大）
- 传送台
    - 时间结晶 + 硅什么的，可传送小型陆地单位
- 大型传送台
    - 时空合金什么，可传送大型陆地单位
- 传送门（仅仅有个概念，可能实现不了）
    - 传送任意单位
    - 入口出口都由两个节点组成，两入口节点连接为一个入口，两出口节点连接为一个出口
    - 一个入口连一个出口
    - in +                  out +
    -    |                      |
    - -> | ---------------------> unit
    -    |                      |
    -    +                      +
- 反射护盾
    - 空间结晶什么的，在护盾基础上加上偏转子弹的功能，血量略弱
    - 加时间流加速贼屌
    - 加时空合金变大变硬也好屌
- 次元核心
    - 5x5 或者 6x6
    - 只有存在次元核心时才能造次元科技线的东西（传送桥除外）
